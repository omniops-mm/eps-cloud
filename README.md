**Executive Productivity System - A Productivity Framework - pre-v0.1.**

This is a long horizon project so I am expecting there to be a lot of changes as I go along. 

As a summary, this is a programmed version of the AI based productivity system I built for myself that got me through some of the busiest periods in my life (including now). The high-level idea is that the system is designed to basically set up your day and the tasks you need to do during that day for you. The intent is to reduce the amount of executive decision making required by training oneself to habitify just following what the system produces as tasks.

The technical aim is a deployable webapp that handles streaks, schedules daily tasks, calendar integration, weather reports and logs habits.

Status: design phase. No code yet.
